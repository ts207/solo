# Cleaned Bars Data (1 Year)

This directory contains 1 year of cleaned 5-minute bar data (OHLCV) for BTCUSDT and ETHUSDT (Perpetual).

## Structure
- `perp/<SYMBOL>/bars_5m/year=2024/`: Partitioned Parquet files by month.

## Usage
These files are provided for offline research and as a sample dataset.
